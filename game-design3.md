📑 GAME DESIGN DOCUMENT (GDD) - Simple MVP EditionProject Working Title: My Tiny Claw MachineGenre: Simple Physics Simulator / CollectionPlatform: PC (Windows)Target Engine: Unity (C#)Scope: Offline Single Player (เล่นคนเดียว 100% ไม่ต้องใช้เซิร์ฟเวอร์)1. Core Gameplay Loop (วงจรการเล่นแบบง่าย)[ หยอดเหรียญ (หักแต้ม UI) ] ➔ [ คุมขาคีบสู้ฟิสิกส์ ] ➔ [ คีบตุ๊กตาตกท่อ ] ➔ [ ของเข้าคลัง / เอาไปตั้งวางบนชั้นโชว์ ]
*หากเงินหมด ➔ กดปุ่มทำความสะอาดร้าน (ค้างไว้ 3 วินาที) ➔ ได้เงินประทังชีวิตกลับมาหยอดใหม่*
2. ฉากในเกม (Game Scenes) - มีแค่ 2 ฉากเท่านั้น!ฉากที่ 1: หน้าตู้คีบ (The Claw Arcade)หน้าจอ: มีตู้คีบ 1 ตู้ตั้งอยู่กลางจอ และมีโมเดลตุ๊กตา/กล่องสุ่ม วางกองอยู่ก้นตู้มุมกล้อง: ล็อกมุมมองไว้ที่หน้าตู้ (สามารถกดปุ่มสลับไปดูมุมมองด้านข้างตู้ได้ เพื่อให้เล็งพิกัดลึก-ตื้นได้แม่นยำขึ้น)ระบบควบคุม:กดปุ่มลูกศร (Arrow Keys) เลื่อน ซ้าย-ขวา-หน้า-หลังกด Spacebar เพื่อทิ้งตัวลงไปคีบวัตถุฉากที่ 2: ห้องโชว์ของสะสม (The Tiny Room)หน้าจอ: ห้องสี่เหลี่ยมเรียบง่าย มีชั้นวางของ 3 ชั้นตั้งอยู่ระบบทำงาน: มีเมนู UI แสดงรายการตุ๊กตาทั้งหมดที่เราคีบได้ เมื่อผู้เล่นกดเลือกชิ้นไหน ตุ๊กตาชิ้นนั้นจะไปโผล่วางบนชั้นโชว์ทันที (เป็นการจัดระเบียบคอลเลกชันแบบง่าย ไม่ต้องใช้ระบบฟิสิกส์จัดวางให้ปวดหัว)3. ระบบฟิสิกส์แกนหลักที่ต้องโฟกัส (Core Mechanics)เพื่อความง่ายและสมจริงในเวลาเดียวกัน ให้ปรับเซ็ตติ้งใน Unity ดังนี้:ตุ๊กตา: ใช้โมเดล 3D สำเร็จรูปทรงกลมหรือทรงเหลี่ยมมนๆ ตั้งค่าตู้ให้วัตถุมี Rigidbody และใช้ Convex Mesh Colliderขาคีบ: ใช้ระบบอนิเมชั่น (Animation) หรือสคริปต์ C# คุมขากรรไกรให้หุบเข้าหากันธรรมดา แต่ตั้งค่า Physic Material ของตัวขาคีบให้มีค่า Friction (แรงเสียดทาน) ต่ำ เพื่อจำลองความลื่นของขาตู้คีบจริงท่อรับของ (The Chute): ใช้กล่องโปร่งใส (Cube) ซ่อนไว้ที่ก้นท่อ ติ๊กช่อง Is Trigger ใน Collider เมื่อวัตถุตุ๊กตาหล่นลงมาแตะกล่องนี้ ให้เพิ่มจำนวนตุ๊กตาในคลังของผู้เล่น แล้วทำลาย (Destroy) วัตถุชิ้นนั้นออกจากฉาก4. แผนงานระยะสั้น (Short-term Action Plan)สัปดาห์ที่ 1 (เซ็ตตู้และขาขยับ): สร้างตู้คีบโหลๆ ขึ้นมา 1 ตู้ เขียนโค้ดควบคุมให้ขาคีบเดินหน้า-ถอยหลัง และกดปุ่มดิ่งลงไปชนพื้นได้สำเร็จสัปดาห์ที่ 2 (ระบบหนีบและตกท่อ): เขียนโค้ดให้ขากรรไกรหุบหนีบวัตถุ ยกขึ้น แล้วเลื่อนกลับมาที่มุมท่อเพื่อปล่อยวัตถุลงกล่องรับของสัปดาห์ที่ 3 (ระบบเงินและฉากโชว์): ทำหน้าจอ UI หักเงินเมื่อกดเล่น ทำปุ่มปั๊มเงินประทังชีวิต และทำหน้าต่างเมนูแสดงตุ๊กตาที่คีบได้บนชั้นวาง
